Что реализовано в коде
Логика игры: Полный цикл от расстановки кораблей до определения победителя. Написаны функции для проверки попаданий, потоплений и промахов.

Логика бота: Прописан алгоритм, по которому компьютер ищет корабли игрока. Бот не просто стреляет наугад, а «добивает» раненый корабль по соседним клеткам.

Использование OpenCV: Библиотека используется для анализа поля. Она проверяет, правильно ли расставлены корабли (соблюдаются ли расстояния в одну клетку между ними и соответствуют ли размеры кораблей правилам).

Консольный ввод: Управление идет через ввод координат в формате Буква-Цифра (A1, J10 и т.д.).

Использованные библиотеки
opencv-python (cv2) — для обработки матрицы поля.

numpy — для создания сеток 10х10 и работы с массивами.
